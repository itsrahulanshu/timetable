│                                                                              │
│  🚀 DEPLOYMENT STEPS:                                                        │
│                                                                              │
│  STEP 1: Get Anti-Captcha API Key                                           │
│  ├─ Visit: https://anti-captcha.com                                         │
│  ├─ Sign up for free account                                                │
│  ├─ Add $5 credit (lasts several months)                                    │
│  └─ Copy API key from Settings → API Setup                                  │
│                                                                              │
│  STEP 2: Push to GitHub                                                     │
│  ├─ Create new repository on GitHub                                         │
│  ├─ Run: git init                                                           │
│  ├─ Run: git add .                                                          │
│  ├─ Run: git commit -m "Initial commit"                                     │
│  ├─ Run: git remote add origin YOUR_REPO_URL                                │
│  └─ Run: git push -u origin main                                            │
│                                                                              │
│  STEP 3: Deploy to Vercel                                                   │
│  ├─ Visit: https://vercel.com                                               │
│  ├─ Sign in with GitHub                                                     │
│  ├─ Click "New Project"                                                     │
│  ├─ Import your repository                                                  │
│  └─ Add environment variables (see below)                                   │
│                                                                              │
│  STEP 4: Add Environment Variables in Vercel                                │
│  ├─ LPU_USERNAME = your_registration_number                                 │
│  ├─ LPU_PASSWORD = your_password                                            │
│  └─ ANTICAPTCHA_API_KEY = your_api_key                                      │
│                                                                              │
│  STEP 5: Click Deploy! 🎉                                                   │
│  └─ Your app will be live at: https://your-app.vercel.app                   │
│                                                                              │
